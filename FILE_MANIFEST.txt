📝 MODIFICATION GUIDE

For Quick Changes:
  1. UI look & feel → edit renderer/style.css
  2. UI behavior → edit renderer/renderer.js
  3. Download settings → edit config.js
  4. App name → edit package.json

For Advanced Changes:
  1. New IPC methods → edit main.js + preload.js
  2. New formats → edit services/downloader.js
  3. Installer config → edit package.json (build section)
  4. Security settings → edit main.js (webPreferences)

For Deployment:
  1. Custom icon → replace assets/icon.png
  2. Version bump → edit package.json
  3. App branding → edit package.json + renderer/style.css
  4. Installer → configure package.json (build section)
